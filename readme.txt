HW1 - In-Memory vs. Disk Based Searching
Unity ID - asarang

assn_1 - the executable.
assn_1.c - the source code for the program

---------
To run:

./assn_1 --mem-lin key.db seek.db
./assn_1 --mem-bin key.db seek.db
./assn_1 --disk-lin key.db seek.db
./assn_1 --disk-bin key.db seek.db

where,
	> --mem-lin, --mem-bin, --disk-lin & --disk-bin are the operating modes.
	> key.db and seek.db are the 2 binary files
